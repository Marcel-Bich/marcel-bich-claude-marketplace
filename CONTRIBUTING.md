Pull Requests sind willkommen, werden jedoch nicht automatisch zusammengeführt. Der Projektinhaber kann Ideen, Ansätze oder Code aus Pull Requests nach eigenem Ermessen in das Projekt aufnehmen. Ursprüngliche Pull Requests können ohne Zusammenführung geschlossen werden. Dies ist kein Urteil über die Qualität des Beitrags.

Der Projektinhaber ist nicht verpflichtet, Einreichungen zu prüfen, darauf zu antworten oder Rückmeldungen zu geben.
