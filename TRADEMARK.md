# Trademark Policy

This document explains the trademark policy for the marcel-bich-claude-marketplace project.

## What This Policy Covers

The following are considered trademarks and branding assets of this project:

- The project name "marcel-bich-claude-marketplace"
- The repository banner images located in `assets/`
- Any logos, icons, or visual branding associated with this project

## Code License vs. Trademark Rights

The source code is licensed under the MIT License. This means you can freely use, modify, and distribute the code.

However, the MIT License does not grant any rights to use the project name, logos, or branding. Trademark rights are separate from copyright and software licenses.

## Permitted Uses (Nominative Fair Use)

You may use the project name in the following ways without permission:

- Truthful statements like "based on marcel-bich-claude-marketplace" or "fork of marcel-bich-claude-marketplace"
- References in documentation, articles, or discussions about this project
- Attribution in derivative works as required by the MIT License
- Linking to this repository

## Prohibited Uses

You may not use the project name, logos, or branding in ways that:

- Suggest endorsement, sponsorship, or certification by this project
- Suggest official affiliation with this project when none exists
- Create confusion about whether your project is the official version
- Imply that your derivative work is an official release

## Requirements for Forks and Derivative Works

If you fork or create a derivative work:

1. Choose a different name for your project
2. Create your own logo and branding
3. Include a clear statement that your project is "not official" and "not affiliated with" the original project
4. Do not use the banner images from `assets/` for your project branding

## Domains, Social Media, and App Stores

Do not register:

- Domain names that are confusingly similar to this project name
- Social media handles that could be mistaken for official accounts
- App store listings that suggest official affiliation

## Branding Assets

The following assets are part of the project branding:

- `assets/repo-banner.jpg`
- `assets/repo-banner_sm1.jpg`

### Permitted use of branding assets

You may use these assets to:

- Link to or reference this repository
- Present, review, or discuss this project
- Provide attribution with a link to the original repository

### Prohibited use of branding assets

You may not:

- Use these assets as branding for your own project or fork
- Remove or obscure the connection to the original project
- Present these assets as your own work

## Permission Requests

If you have a use case not covered by this policy or want explicit permission for a specific use, contact the project maintainer by opening an issue in this repository with the subject "Trademark Permission Request".

Include in your request:

- Your intended use
- Context and purpose
- How you plan to make clear that your use is not an official endorsement

## Enforcement

This policy exists to protect users from confusion about what is and is not an official part of this project. Violations may result in requests to cease the infringing use.
